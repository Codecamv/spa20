# PWA

We're going to make use of the PWA features.

- manifest.json
  - "background_color": "#fff",
  - "theme_color": "#FFC107"
- index.html
  - <meta name="theme-color" content="#FFC107">
  - <meta name="msapplication-TileColor" content="#FFC107">
- npm run build
- npm install -g serve
- serve --port 5001 dist/
